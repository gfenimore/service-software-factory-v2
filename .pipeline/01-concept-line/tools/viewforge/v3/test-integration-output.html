ViewForge Integration Support - Test Output

Test Case 1: Field with Address Validation

This demonstrates a service address field with address-validation integration:

<div class="field-group" 
     data-entity="serviceLocation" 
     data-field="address" 
     data-type="string"
     data-integration="address-validation"
     data-integration-required="true"
     data-integration-capabilities="autocomplete,validation,geocoding">
    <label>Service Address</label>
    <input type="text" name="address" />
</div>

Test Case 2: Map Display with Dependencies

This shows a geographic display that depends on the address field:

<div class="section-map"
     data-entity="serviceLocation"
     data-section="territoryMap"
     data-integration="geographic-display"
     data-depends-on="address"
     data-integration-required="false"
     data-integration-capabilities="markers,clustering,routing,territories">
    <!-- Map display placeholder -->
    <div class="map-placeholder">[Map Display]</div>
</div>
